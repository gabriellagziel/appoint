### Prerequisites

- Dart SDK 3.4.0 or higher

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/gabriellagziel/appoint.git
   cd appoint
   ```

2. Install dependencies:
   ```bash
   flutter pub get
   ```

3. Run the application:
   ```bash
   flutter run
   ```
